# Specification

## Summary
**Goal:** Refresh the frontend visual theme of the AI Media Studio with a mature gradient glassmorphism design system.

**Planned changes:**
- Replace all page backgrounds with CSS linear gradients using deep charcoal, slate, and warm amber/emerald/gold accent tones (no blue or purple)
- Apply glassmorphism styling to cards and modals: backdrop-blur, semi-transparent backgrounds, and subtle glowing borders
- Update accent colors (amber, emerald, or gold) consistently across buttons, badges, and interactive elements
- Ensure sharp, high-contrast typography against dark gradient backgrounds
- Add neon-glow or gradient highlight hover/focus effects to buttons and cards
- Apply consistent theme across IndexPage, GeneratePage, GalleryPage, CreditsPage, and AdminPage

**User-visible outcome:** The entire app displays a polished, mature dark gradient aesthetic with glassmorphism surfaces and warm accent tones, giving a premium look across all pages.
